# task-buddy

## Client side
### Technologies used
React, TailwindCSS

### Run client side
1. 'cd' into /client
2. Run 'npm install'
3. Run 'npm start'


## Server side
### Technologies used
Express, Sequelize, and PostgreSQL as the database

### Run client side
1. 'cd' into /server
2. Run 'npm install'
3. Run 'npm start'
